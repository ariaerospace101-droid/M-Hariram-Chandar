# M-Hariram-Chandar

**Tagline:** M-Hariram-Chandar — high-performance aerospace controls & simulation utilities

## Ad / Short blurb

M-Hariram-Chandar is a compact library of aerospace control algorithms, simulation helpers, and demo projects designed for rapid prototyping and research. Built for performance and clarity — drop it into your simulation pipeline to accelerate development.

## Introduction

M-Hariram-Chandar provides reusable components for flight dynamics, control laws, sensor fusion, and simulation tooling. It is geared toward developers and researchers working on UAV and spacecraft control, guidance, and simulation.

## Key features

- Modular control algorithms (PID, LQR, adaptive controllers)
- Simulation utilities for dynamics and sensor models
- Example demonstrations and small datasets
- Clear, commented code aimed at education and fast integration
